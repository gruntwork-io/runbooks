Runbooks is a desktop app for working with runbooks. You can launch it by double-clicking the app icon or via the CLI. All CLI commands accept a local path or a remote URL as the runbook source.

- [`runbooks open`](/commands/open) -- Open a runbook in the desktop app
- [Watch mode](/commands/watch) -- Toggle live-reload for authoring runbooks

### Launching the app

You can launch a runbook by double-clicking the Runbooks app, or from the command line:

```bash
runbooks open ./path/to/runbook
```

### Remote URLs

You can open a runbook directly from a remote URL via the CLI, removing the need to manually download any runbook files.

```bash
runbooks open https://github.com/org/repo/tree/main/runbooks/launch-rds
```

See [`runbooks open`](/commands/open/#remote-sources) for supported URL formats and authentication for private repos.
